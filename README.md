# RescueHope — HTML Template

RescueHope is a Local Non-Profit / Animal Rescue HTML Template designed to drive donations, volunteer sign-ups, and transparently showcase impact. 

## Documentation
Please view the full documentation at `documentation/README.md`.

## Features
- Fully responsive mobile-first design.
- Dark mode and RTL support.
- Tailwind CSS included.
- Minimal dependencies and clean architecture.

## Installation
Just open `index.html` in your favorite browser. No build tools are required unless you want to customize Tailwind CSS outside of the script tag.

Refer to `documentation/README.md` for full details on support and customization.
