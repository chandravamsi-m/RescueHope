## Installation
1. Extract the template folder.
2. Open `index.html` in your browser.
3. To customize the design, modify the CSS files in `assets/css/`.

## Customization
- **Colors**: Modify the `tailwind.config` in the `<head>` of HTML files.
- **Images**: Replace images in `assets/images/`.
- **CSS**: Edit `assets/css/style.css`.
- **Dark Mode**: Modify `assets/css/dark-mode.css`.
- **RTL**: Modify `assets/css/rtl.css`.

## Page Structure
- **assets/**: Contains CSS, JS, images, and fonts.
- **pages/**: Contains various content pages like About, Services, Donate, etc.
- **auth/**: Contains authentication pages like Login and Register.
- **index.html**: Main homepage.
