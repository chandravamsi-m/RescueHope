# Installation Guide

## Getting Started
Welcome to the RescueHope HTML Template! This template is built with standard HTML, CSS (via Tailwind CSS CDN or compiled local styles), and Vanilla JavaScript.

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor or IDE (like VS Code, Sublime Text, or WebStorm) if you plan to edit the code.

### Step-by-Step Installation
1. **Extract the Files:** 
   Unzip the downloaded package to your desired directory.
2. **Open the Project:**
   Open the unzipped folder in your code editor.
3. **Run Locally:**
   Simply double-click on `index.html` to open it in your browser. 
   
   *Tip:* For the best experience and to avoid local file parsing issues with modules (CORS), we highly recommend running a local server. If you use VS Code, you can install the **Live Server** extension, right-click on `index.html`, and select "Open with Live Server".

## Deployment
Since this is a static HTML template, it can be deployed on any standard web hosting service or static host provider:
- GitHub Pages
- Netlify
- Vercel
- Standard cPanel / FTP Server (upload the contents of the root directly to your `public_html` directory).
