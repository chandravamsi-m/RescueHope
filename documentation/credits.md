RescueHope is built with standard Open-Source and freely available tools. 

## Libraries & Frameworks
* **Tailwind CSS (CDN)** - A utility-first CSS framework for rapid UI development.
* **Vanilla JavaScript** - Ensuring maximum performance without jQuery overhead.

* **Icons:** [Iconify](https://iconify.design/) loaded via script for fast, lightweight vector icons.
* **Fonts:** *Outfit* font family via Google Fonts. (Open Font License)
* **Images:** All placeholder images are royalty-free. When taking this site to production, it is highly recommended to replace these placeholders with photos of actual animals from your rescue network.
